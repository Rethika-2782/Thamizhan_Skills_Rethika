# Student Registration Form

## How to use
1. Open `index.html` in any modern browser (Chrome, Edge, Firefox).
2. Fill out the form and click **Register**.
3. Inline validation errors will appear in red; a success message will appear in green when all fields are valid.

## Validation rules
- **Name**: required (at least 2 characters after trimming)
- **Email**: must match a basic email format
- **Phone**: exactly **10 digits** (non-digits are ignored for validation)
- **Password**: minimum **6 characters**
