## Thamizhan Skills Landing Page

Minimal, modern startup-style landing page built with **HTML + CSS + JavaScript** (single file design).

### What you get
- Sticky navbar with subtle shadow
- Hero section with premium blue-to-purple gradient
- About section (mission/story text)
- Services section (cards with icons + hover lift + shadow)
- Contact section (styled form with front-end validation + success message)
- Dark footer with white text

### Tech stack
- HTML (semantic sections)
- CSS (responsive layout, smooth scrolling, hover animations)
- JavaScript (mobile menu toggle + contact form validation)

### How to run
1. Open `index.html` in your browser:
   - `landing page/index.html`
2. Use the navbar to jump between `Home`, `About`, `Services`, and `Contact`.

### Contact form behavior
The form performs **front-end validation only**:
- Name: required (min 2 chars)
- Email: required (simple email regex)
- Message: required (min 10 chars)
- On success: a message is shown and the form resets

### Customization ideas
- Replace company text (hero/about/services/contact details) directly in `index.html`.
- Update service titles/descriptions in the Services cards.
